Breackpoints
es una manera en la cual bootstrap a partir de un tamaño de pantalla, nosotro podemos cambiar los estilos de css y esto lo podemos hacer a travez de media jquery

--------estilos a travez de media jquery
        @media (min-width: 576px) {//Tamaño minimo de pantalla en bootstrap//CELULARES
                @media (min-width: 768px)//TABLETS
        @media (min-width: 992px) {//pantalla grande
        @media (min-width: 1200px) {//pantalla extra grande


---------------otro ejemplo con media jquery
      <style media="screen">
        @media (max-width: 1200px) {
          body {
            background: red;
          }
        }

        @media (max-width: 992px) {
          body {
            background: blue;
          }
        }

        @media (max-width: 768px) {
          body {
            background: green;
          }
        }

        @media (max-width: 575px) {
          body {
            background: yellow;
          }
        }
      </style>
